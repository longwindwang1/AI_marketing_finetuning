# 训练数据生成 Prompt

## 使用方法
1. 将下面的 Prompt 复制到 Claude / GPT 中
2. 每次生成 20 条，分 10-15 次生成（覆盖不同类别和行业）
3. 将每次生成的 JSON 内容追加到 `data/raw/generated_data.jsonl`（每行一个 JSON 对象）
4. 最后运行：`python data/prepare_dataset.py process --input data/raw/generated_data.jsonl --output data/processed`

---

## Prompt 1：投放数据分析（生成 3-4 次，换不同行业）

```
你是短视频广告投放领域的数据标注专家。请生成 20 条「投放数据分析」类型的训练样本。

格式要求：
- 输出 JSON 数组，每条包含 "instruction" 和 "output" 两个字段
- instruction：用户提供一组广告投放数据，请求分析和优化建议
- output：专业的分析报告，包含指标诊断、问题定位、优化建议

覆盖要求（本次请覆盖以下行业，每个行业 4 条）：
- 美妆个护
- 食品饮料
- 在线教育
- 游戏
- 本地生活

平台覆盖：抖音、快手、TikTok、视频号、B站 混合使用

每条 instruction 必须包含具体数据（消耗、展示量、点击量、转化量、CPM、CTR、CVR、CPA、ROI 等），数字要合理。

每条 output 要求：
1. 用表格对比行业基准
2. 明确标注哪些指标正常/偏低/偏高
3. 给出 3-5 条具体可执行的优化建议
4. 长度 400-800 字

直接输出 JSON 数组，不要其他解释。
```

## Prompt 2：流量诊断（生成 2-3 次，换不同问题场景）

```
你是短视频广告投放领域的数据标注专家。请生成 20 条「流量诊断」类型的训练样本。

格式要求：
- 输出 JSON 数组，每条包含 "instruction" 和 "output" 两个字段
- instruction：用户描述广告投放中遇到的异常现象
- output：专业的诊断分析和解决方案

异常场景覆盖（每种 3-4 条）：
- 消耗突然下降 / 不起量
- CTR 持续走低
- CVR 下降但 CTR 正常
- CPA 飙升
- 新计划跑不动
- 账户被限流 / 审核问题

每条 output 要求：
1. 按概率排序列出可能原因（3-5 个）
2. 每个原因说明判断方法
3. 给出紧急措施 + 短期优化 + 长期机制
4. 长度 400-800 字

平台混合使用：抖音、快手、TikTok、视频号
直接输出 JSON 数组，不要其他解释。
```

## Prompt 3：素材/创意分析（生成 2-3 次，换不同产品类型）

```
你是短视频广告投放领域的数据标注专家。请生成 20 条「素材创意分析」类型的训练样本。

每条 output 要求：
1. 给出整体评分（满分10分）
2. 逐段分析优劣
3. 指出前3秒钩子的问题
4. 给出完整的优化脚本
5. 长度 400-800 字
```

## Prompt 5：ROI 优化（生成 2-3 次，换不同场景）

```
你是短视频广告投放领域的数据标注专家。请生成 20 条「ROI 计算与优化」类型的训练样本。

格式要求：
- 输出 JSON 数组，每条包含 "instruction" 和 "output" 两个字段
- instruction：用户提供投放数据（含消耗、GMV、退货率、成本率等），请求 ROI 分析
- output：精确的 ROI 计算 + 盈亏分析 + 优化策略

场景覆盖：
- 直播带货 ROI 优化（5条）
- 信息流广告 ROI 优化（5条）
- 多平台预算分配（4条）
- 新品冷启动 ROI 预估（3条）
- 大促期间 ROI 波动分析（3条）

每条 output 要求：
1. 详细的数学计算过程（表面ROI → 扣退货 → 净利润）
2. 标注盈亏平衡点
3. 给出多维度优化策略（退货率、客单价、CPA、人群等）
4. 用表格展示优化前后对比
5. 长度 400-800 字

直接输出 JSON 数组，不要其他解释。
```

## 数据合并说明

每次 Claude/GPT 返回 JSON 数组后，将数组中的每个对象单独放一行，追加到同一个文件：

```
data/raw/generated_data.jsonl
```

格式示例（每行一个对象）：
```json
{"instruction": "请分析...", "output": "## 分析报告\n..."}
{"instruction": "我的广告...", "output": "## 诊断\n..."}
```

全部生成完后运行：
```bash
python data/prepare_dataset.py process --input data/raw/generated_data.jsonl --output data/processed
```
